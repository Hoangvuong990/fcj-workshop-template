**Trong trang này** bạn sẽ cần giới thiệu worklog của bạn **như thế nào**? Bạn hoàn thành chương trình trong vòng **bao nhiêu tuần**? Bạn **đã làm gì** trong các tuần đó?


Thông thường và cũng là tiêu chuẩn, một worklog được thực hiện trong khoảng 3 tháng (trong suốt thời gian thực tập) với nội dung các tuần như sau:

**Tuần 1:** [Làm quen với AWS và các dịch vụ cơ bản trong AWS](1.1-week1/)

**Tuần 2:** [Tìm hiểu AWS Cost Management và thiết lập AWS Budgets](1.2-week2/)

**Tuần 3:** [AWS Support và cơ bản về Mạng trên AWS (VPC, NACL, Security Group, ELB)](1.3-week3/)

**Tuần 4:** [Tìm hiểu dịch vụ máy chủ ảo Amazon EC2, các thành phần bổ trợ và Auto Scaling](1.4-week4/)

**Tuần 5:** [Lưu trữ Amazon S3, giải pháp Hybrid Storage và quản lý danh tính AWS IAM](1.5-week5/)

**Tuần 6:** [Tìm hiểu Cơ sở dữ liệu AWS (RDS, Aurora, Redshift), AWS Organizations và Security Hub](1.6-week6/)

**Tuần 7:** [Khởi động dự án Wakan, thiết lập môi trường và vẽ wireframe giao diện trên Google Drawings/Jamboard](1.7-week7/)

**Tuần 8:** [Đồng bộ API contract, dựng UI tĩnh và tích hợp xác thực Amazon Cognito](1.8-week8/)

**Tuần 9:** [Hoàn thiện UI chọn lựa, tích hợp API Gateway và lập trình các trạng thái tương tác](1.9-week9/)

**Tuần 10:** [Thiết kế giao diện Timeline lịch trình, tích hợp bản đồ và tối ưu Responsive](1.10-week10/)

**Tuần 11:** [Tinh chỉnh UI/UX, thực hiện kiểm thử và sửa lỗi giao diện](1.11-week11/)

**Tuần 12:** [Chạy thử nghiệm hệ thống, tối ưu hóa lỗi và hoàn thiện báo cáo kỹ thuật](1.12-week12/)
