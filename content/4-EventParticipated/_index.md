Throughout my internship with the First Cloud Journey program, I had the privilege of attending 3 major tech events. Each of these gatherings provided a great mix of technical insights, professional development, and networking opportunities with fellow peers.

### [Event 1: AWS FIRST CLOUD AI JOURNEY COMMUNITY DAY](4.1-Event1/)  
&emsp;**Event Name:** AWS FIRST CLOUD AI JOURNEY COMMUNITY DAY  

&emsp;**Date & Time:** 09:00, May 23, 2026  

&emsp;**Location:** 26th Floor, Bitexco Tower, 02 Hai Trieu Street, Saigon Ward, Ho Chi Minh City  

&emsp;**Role:** Attendee  

&emsp;**Brief Description:** A detailed conference highlighting edge security via CloudFront, LLM inner workings, agentic workflows using Amazon Bedrock, and streamlined processes for AI product development.

&emsp;**Value Gained:** Acquired practical insights on launching enterprise cloud architectures and understood the critical need for robust network boundaries when deploying AI applications.
